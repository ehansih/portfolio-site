# Portfolio Site

Personal portfolio of **Harsh Vardhan Singh Chauhan** — Telecom Security specialist, 5G Architect, and Product Strategist with 16+ years of experience.

## Live Site
🌐 [saharsh-projects.vercel.app](https://saharsh-projects.vercel.app)

## About
Dark terminal-aesthetic portfolio built from scratch with zero frameworks. Features a custom animated cursor, scroll-reveal animations, and interactive sections.

## Tech Stack
- **HTML5** — Semantic structure
- **CSS3** — Custom properties, Grid, Flexbox, animations
- **Vanilla JavaScript** — No frameworks, no dependencies
- **Vercel** — Deployment

## Features
- Custom animated cursor with ring follower
- Particle canvas background
- Scroll-reveal animations (Intersection Observer)
- Responsive mobile layout
- Photography lightbox gallery
- GitHub projects showcase (all 9 repos)
- Contact form via Formspree
- Newsletter subscribe via Mailchimp
- Domain expertise cards
- Career timeline
- Philosophy section

## Sections
- Hero / Landing
- Domains of Mastery
- Career Journey (16+ years)
- Philosophy
- GitHub Projects
- Photography
- Contact

## Deployment

```bash
# Install Vercel CLI
npm i -g vercel

# Deploy
vercel --prod
```
